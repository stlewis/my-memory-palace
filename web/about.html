About My Memory Palace

This page is mostly about the project that is MyMemoryPalace, rather than the application itself. If you want to know more about the application or the memory palace technique, you should check out this page.

MyMemoryPalace may be a cool idea, but it's main purpose for existing is to give me an opportunity to try developing a web application using React as the front-end component. The "informational" pages are just HTML with some Bootstrap thrown in for good measure, but the training application itself was created using React components.

This is my first foray in to React development, and I'm sure I'm doing a hundred things wrong, but it is "working" in some sense, so I'm keeping it around, if for nothing else than to serve as a reminder of what not to do to a later version of myself.

If you're curious how I did it, or you'd like to download this entire site as something to play around with yourself, feel free to clone it from Github. Issues, comments and so on are always welcome there as well.
